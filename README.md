# Snowpeak Mountain Lodge

A GitHub-ready static website inspired by the structure of the provided chapter project, redesigned around snowy mountaintop scenery.

## Files included
- `index.html`
- `contact.html`
- `styles.css`
- `stays/` multi-page room section
- `adventures/` multi-page activity section
- `images/` local image assets

## How to post on GitHub Pages
1. Create a new repository.
2. Upload every file and folder exactly as they appear here.
3. Make sure `index.html` stays in the root folder.
4. Enable GitHub Pages in the repository settings.

Because all links are relative and all images are local, the site is ready for GitHub hosting.
